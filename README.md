# gmall
This is a gmall Java project  
**Springboot version: 1.5.21.RELEASE**  
SOA mode is provided by **dubbo and zookeeper**.  
Image storage is provided by **fastdfs1.27 and nginx**.  
Search function is provided by **elasticsearch**.
